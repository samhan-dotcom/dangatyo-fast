💡왜 만들었나

통신사 단가표(LGU+, KT, SKT 등)는 이미지·PDF 형태로 배포되어, 수작업으로 Excel에 옮기는 반복 작업이 발생했습니다.
Claude의 Vision 기능으로 이 과정을 자동화하고, 팀원 누구나 링크 하나로 사용할 수 있게 만든 도구입니다.

Claude 1회 호출 · 약 10~20초 · 50~120원/장

PaddleOCR 무게 때문에 로컬 랩탑에서만 실행 가능

PaddleOCR이 너무 무거워 Streamlit Cloud 무료 플랜에서 설치 실패합니다. Vercel·Heroku는 WebSocket 기반 Streamlit과 호환되지 않습니다.

🏗️기술·설계 결정

AI 모델: claude-sonnet-4-6

Opus는 비용이 너무 높고, Haiku는 표 추출 정확도가 낮았습니다. Sonnet이 속도·비용·정확도의 균형점입니다.
Extended Thinking (adaptive, effort: low) 적용으로 숫자 판독 정확도를 높이면서 비용을 최소화했습니다.

🔑API 키 변경

API 키는 console.anthropic.com/settings/keys에서 발급·확인합니다.
형식은 sk-ant-api03-...으로 시작합니다.

① 로컬 실행 시 (.env 파일)

단가표앱/ 폴더 안의 .env 파일을 메모장으로 열어서 수정합니다.

파일: 단가표앱/.env
ANTHROPIC_API_KEY=sk-ant-api03-새로운키를여기에붙여넣기
⚠️ .env 파일은 .gitignore에 등록되어 GitHub에 올라가지 않습니다. 절대 직접 GitHub에 커밋하지 마세요.

② Streamlit Cloud 배포 앱

  1. 1
    share.streamlit.io 로그인 samhan-dotcom 계정으로 로그인합니다.
  2. 2
    dangatyo-fast 앱 → 우측 점 메뉴 → Settings 앱 목록에서 해당 앱을 찾아 설정으로 진입합니다.
  3. 3
    Secrets 탭 → 내용 수정 → Save 아래 형식으로 입력하고 저장하면 1분 내 적용됩니다.
    ANTHROPIC_API_KEY = "sk-ant-api03-새로운키"
✅ Secrets는 암호화되어 저장되며 코드에 노출되지 않습니다. 팀원도 키를 볼 수 없습니다.

③ API 키 만료·오류 시 증상

🚨 앱에서 ⚠️ ANTHROPIC_API_KEY 미설정 메시지가 뜨거나, 추출 중 API 오류: 401이 나오면 키가 만료되었거나 잘못 입력된 것입니다. 위 방법으로 새 키로 교체하세요.

🤖AI 모델 변경

더 저렴하거나 더 정확한 새 모델로 교체하고 싶을 때 app_fast.py 상단 두 곳을 수정합니다.

# app_fast.py 상단 (약 38~45번째 줄)

MODEL = "claude-sonnet-4-6"   # ← 여기를 새 모델명으로 교체
MAX_TOKENS = 16000

PRICES = {
    "claude-opus-4-7":   (5.0, 25.0),   # (입력, 출력) $/100만토큰
    "claude-sonnet-4-6": (3.0, 15.0),   # ← 새 모델 단가도 여기에 추가
}
모델입력 단가출력 단가특징
claude-haiku-4-5$0.8$4.0가장 저렴, 정확도 낮음
claude-sonnet-4-6$3.0$15.0현재 사용 중 균형
claude-opus-4-7$5.0$25.0가장 정확, 가장 비쌈
ℹ️ 최신 모델명은 docs.anthropic.com/en/docs/about-claude/models에서 확인할 수 있습니다.
모델 변경 후 git push하면 Streamlit Cloud에 자동 반영됩니다.

📋템플릿 추가 방법

templates.py 파일만 수정하면 새 통신사·양식을 추가할 수 있습니다. 앱 코드(app_fast.py)는 건드리지 않아도 됩니다.

현재 등록된 양식

키표시 이름설명
auto자동 감지양식 모를 때 사용, prompt=None
lgup📱 LGU+LGU+ 전용 컬럼 지정
kt📱 KTKT 전용 컬럼 지정
skt📱 SKTSKT 전용 컬럼 지정

새 양식 추가 예시

# templates.py

TEMPLATES = {
    "auto": { ... },  # 기존 내용 유지
    "lgup": { ... },

    # 👇 여기에 새 항목 추가
    "mvno": {
        "display_name": "📡 알뜰폰 (MVNO)",
        "prompt": """이 이미지는 알뜰폰 단가표입니다.
아래 컬럼을 정확히 추출해주세요:
- 통신사 (망): LGU+/KT/SKT 중 하나
- 요금제명
- 월 이용료 (VAT 포함)
- 기본 데이터
- 음성통화
- 특이사항

숫자는 본 대로 정확히 입력하고, 확신 낮은 셀은 uncertain_cells에 표시하세요.""",
    },
}
  1. 1
    templates.py를 텍스트 편집기로 열기 VS Code, 메모장 등 아무 편집기 사용 가능
  2. 2
    TEMPLATES 딕셔너리에 새 항목 추가 위 예시 형식을 참고해 키, display_name, prompt 작성
  3. 3
    저장 후 git push 
    git add templates.py
git commit -m "알뜰폰 양식 추가"
git push
  4. 4
    Streamlit Cloud 자동 재배포 (약 1~2분) 앱을 새로고침하면 드롭다운에 새 양식이 나타납니다.
💡 prompt 작성 팁 — 추출하고 싶은 컬럼명을 구체적으로 나열할수록 정확도가 높아집니다. 처음에는 자동감지로 한 번 추출해보고, 자주 틀리는 패턴을 prompt에 명시하는 방식으로 개선하면 됩니다.
