💡왜 만들었나

통신사 단가표(LGU+, KT, SKT 등)는 이미지·PDF 형태로 배포되어, 수작업으로 Excel에 옮기는 반복 작업이 발생했습니다.

Claude의 Vision 기능으로 이 과정을 자동화하고, 팀원 누구나 링크 하나로 사용할 수 있게 만든 도구입니다.

🔀버전 구성

⚡ 빠른 버전 app_fast.py

Claude 1회 호출 · 약 10~20초 · 50~120원/장
Streamlit Cloud에 배포되어 링크만으로 팀 전체 사용 가능

🔬 정밀 버전 app.py

Claude 2~4회 병렬 호출 + PaddleOCR 교차검증 · 약 30~60초 · 100~250원/장
PaddleOCR 때문에 로컬 랩탑에서만 실행 (클라우드 배포 불가)

ℹ️ 왜 정밀 버전을 클라우드에 못 올리나?
PaddleOCR이 너무 무거워 Streamlit Cloud 무료 플랜에서 설치 실패합니다. Vercel·Heroku는 WebSocket 기반 Streamlit 자체와 호환되지 않습니다.

AI 모델: claude-sonnet-4-6

Extended Thinking (adaptive, effort: low) 적용 — 숫자 판독 정확도 향상, 비용 최소화.

⚠️ Cloudflare Tunnel 주의사항
localhost 대신 127.0.0.1을 사용해야 합니다.
localhost는 IPv6 [::1]로 해석되고, Streamlit은 IPv4만 리스닝해서 연결 실패합니다.

💻로컬 실행

사전 조건

  • Python 3.12 — PaddleOCR 호환 (3.13+ 불가)
  • cloudflared — 다운로드 링크
# 1. 가상환경 생성 (반드시 Python 3.12)
py -3.12 -m venv .venv
.venv\Scripts\activate

# 2. 패키지 설치
pip install -r requirements.txt

# 3. OCR 설치 (정밀 버전 사용 시만)
pip install paddlepaddle paddleocr

# 4. .env 파일 생성
ANTHROPIC_API_KEY=sk-ant-api03-...

실행

# 빠른 버전만 (포트 8502)
streamlit run app_fast.py --server.port 8502

# 두 버전 + Cloudflare 동시 시작 (Windows)
run_all.bat

run_all.bat이 자동으로 처리하는 것:

  • 기존 프로세스 종료
  • 정밀 버전 (포트 8501) 시작
  • 빠른 버전 (포트 8502) 시작
  • Cloudflare Tunnel 2개 시작 → trycloudflare.com URL 출력

☁️Streamlit Cloud 배포

코드 수정 → 배포 워크플로우

# 수정 후 push → Streamlit Cloud 자동 재빌드 (약 1~2분)
git add .
git commit -m "변경 내용 설명"
git push

API 키 설정 위치

Streamlit Cloud → 앱 설정 → Secrets 탭에 아래 형식으로 입력:

ANTHROPIC_API_KEY = "sk-ant-api03-..."
😴 절전 모드 안내 (팀원에게 공유)
무료 플랜은 미접속 시 앱이 잠듭니다. "Zzzz" 화면이 뜨면 파란 버튼 클릭 → 약 1분 대기하면 재시작됩니다.

💰비용 구조

항목단가
claude-sonnet-4-6 입력$3.00 / 100만 토큰
claude-sonnet-4-6 출력$15.00 / 100만 토큰
캐시 쓰기입력의 1.25배
캐시 읽기입력의 0.10배 (90% 할인)
빠른 버전 (장당)약 50~120원
정밀 버전 (장당)약 100~250원

모델 업그레이드 시 app_fast.py 상단 두 곳만 수정합니다:

MODEL = "claude-sonnet-4-6"   # ← 모델명 변경

PRICES = {
    "claude-sonnet-4-6": (3.0, 15.0),  # ← (입력, 출력) $/100만토큰
}

🚀향후 개선 방향

단기

  • 추출 결과 저장·불러오기 (히스토리)

장기

  • 이메일·카카오톡으로 받은 단가표 자동 처리
  • 데이터베이스 연동 (단가표 이력 관리)
